Description

  The SimpleBluetooth4Win C++ library allows to use in an easy way the 
bluetooth stack in the windows operating system, enumerating the paired 
bluetooth devices and writing to and reading from them streams of bytes 
in a straightforward manner through the simplified use of the RFCOMM 
windows bluetooth sockets.
